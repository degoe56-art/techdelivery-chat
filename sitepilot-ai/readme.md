TechDelivery Chat

An AI-powered chatbot plugin for WordPress that answers visitor questions using your own website content.

Features

* 🤖 AI-powered chatbot using OpenAI
* 🔍 Semantic search with embeddings
* 📚 Clickable source references
* ⚡ Incremental indexing
* 📱 Fully responsive chat widget
* 🎨 Customizable title, welcome message and colors
* 🚀 Fast AJAX indexing
* 🌐 Works with any WordPress theme

⸻

Requirements

* WordPress 6.5 or newer
* PHP 8.0 or newer
* OpenAI API key

⸻

Installation

1. Upload the plugin to:

/wp-content/plugins/techdelivery-cat

2. Activate the plugin.
3. Open:

SitePilot AI

from the WordPress admin menu.

4. Enter your OpenAI API key.
5. Start indexing.

The chatbot is now ready to answer questions based on your website.

⸻

How it works

The plugin indexes your website content and splits it into smaller content blocks.

Each content block is converted into an OpenAI embedding and stored locally.

When a visitor asks a question:

1. A semantic search finds the most relevant content.
2. GPT generates a natural language answer.
3. Clickable source references are displayed below the answer.
